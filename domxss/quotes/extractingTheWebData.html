========= Web Crawler: Extracting the Web Data =========

  1. "Web crawling can also be called as a graph search problem as web is considered to be a large graph where nodes are the pages and edges are the hyperlinks."
  2. "Web crawlers can be used in various areas, the most prominent one is to index a large set of pages and allow other people to search this index"
  3. "A list of URLs is available with the web crawler and each URL is called a seed. Each URL is visited by the web crawler. It identifies the different hyperlinks in the page and adds them to the list of URLs to visit. This list is termed as crawl frontier."
